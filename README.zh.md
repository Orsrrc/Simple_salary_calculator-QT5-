# Simple Salary Calculator

软件名称

Simple Salary Calculator

软件概要

键入用户的工资等计算对应月收入

功能实现

-  模块一
  - 计算工资加减项
- 模块二
  - 键入加项减项等信息 
-  模块三
   -  通过键入以上模块的信息计算出月所得工资
